# GIT.md — Commit Conventions

> **Status:** Active

## Commit Conventions

We enforce strict **Conventional Commits**. AI Agents must adhere to this format.

### Format
`type(scope): subject`

### Allowed Types
- **`feat:`** A new component or feature.
- **`fix:`** A bug fix.
- **`docs:`** Changes to context files.
- **`style:`** Design token updates, formatting.
- **`refactor:`** Code changes (e.g. migrating to new Runes syntax).
- **`perf:`** Performance improvements.
- **`chore:`** Package manager updates, configs.
